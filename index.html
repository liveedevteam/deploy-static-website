ข้อมูลยี่ห้อรถ

ขั้นตอนในการทำ ML

1. Load Data Set ของรถยนต์ในรูปของ JSON

2. Set Up ตัววัดผล ในที่นี้จะเลือก อัตราการกินน้ำมัน และ แรงม้า

3. Clean Data ตัวอย่าง

4. Visualization Data ตามกราฟซ้ายมือ

5. สร้าง Model Architecture
โดยใช้หลักการ โมเดลนี้จะใช้อัลกอริธึมใดในการคำนวณคำตอบ
โมเดล ML เป็นอัลกอริธึมที่รับอินพุตและสร้างเอาต์พุต เมื่อใช้โครงข่ายประสาทเทียม อัลกอริธึมคือชุดของชั้นของเซลล์ประสาทที่มี ตัวเลข ควบคุมผลลัพธ์ กระบวนการฝึกอบรมจะเรียนรู้ค่าที่เหมาะสมที่สุดสำหรับตัวเลขเหล่านั้น

6. เตรียมข้อมูลสำหรับใช้ Training ด้วยการทำ Normalize Data
ดูนิยามของ Normalization

7. ด้วยการสร้างอินสแตนซ์โมเดลและข้อมูลที่แสดงเป็นเทนเซอร์ เริ่มการ Training
อัลกอริทิ่มที่เกี่ยวข้อง
- Optimizer มีหน้าที่ ควบคุมการอัปเดตของโมเดลตามที่เห็นตัวอย่าง
- Loss มีหน้าที่ ฟังก์ชันที่จะบอกโมเดลว่าการเรียนรู้แต่ละแบตช์ (ชุดย่อยของข้อมูล) ที่แสดงนั้นทำได้ดีเพียงใด เราใช้ meanSquaredError เพื่อเปรียบเทียบการคาดคะเนของแบบจำลองกับค่าจริง

8. สร้างการทำนาย ตอนนี้โมเดลของเราได้รับการฝึกฝนแล้ว เราต้องการทำการคาดคะเน มาประเมินแบบจำลองโดยดูจากสิ่งที่คาดการณ์สำหรับช่วงตัวเลขที่มีกำลังม้าต่ำถึงสูง
